# School-Student-Registration-Edit-System-

A simple PHP and MySQL based Student Registration System developed for learning CRUD operations and AJAX functionality.


## 🚀 Features
Student Registration Form  
JavaScript Validation  
Save Student Data in MySQL  
Display Registered Students  
View Student Details using AJAX  
Edit and Update Student Details  
First Level responsive UI Design (improvement needed) 


## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- AJAX
- XAMPP


## ⚙ Setup Instructions

### 1. Install XAMPP
Start:
- Apache
- MySQL

### 2. Move Project Folder
Place project inside:
C:\xampp\htdocs\


### 3. Create Database
Database Name:
santaschool_db


### 4. Create students table
Run the SQL query inside phpMyAdmin.


### 5. Run Project
Open browser:
http://localhost/student_registration

---

## 🎯 Functionalities

### Registration Form
Users can register student details.

### Student List
Displays all registered students from database.

### AJAX View
Displays student details dynamically without page reload.

### Edit/Update
Allows modification of existing student records.
